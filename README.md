# OpenUHI Melbourne
### An Open-Source GIS Application 

> An interactive, browser-based web map exploring the spatial relationship between Land Surface Temperature, building typology, and tree canopy coverage across Melbourne, Australia. This tool was designed for Cloud-Based Open-Source GIS at RMIT University to demonstrate a complete three-tier open-source GIS pipeline: PostGIS (data tier) → QGIS + qgis2web (processing and static export) → VS Code (custom Leaflet code) → GitHub Pages (cloud deployment).
